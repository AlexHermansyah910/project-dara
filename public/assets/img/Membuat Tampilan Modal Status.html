Untuk membuat tampilan modal yang menampilkan detail sebuah status, Anda dapat menggunakan HTML, CSS, dan JavaScript. Berikut ini contoh kodingan yang dapat Anda gunakan sebagai referensi:

HTML:

html
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" type="text/css" href="style.css">
</head>
<body>
    <!-- Tombol untuk membuka modal -->
    <button id="openModalBtn">Buka Modal</button>

    <!-- Modal -->
    <div id="infopostingan" class="modal">
        <div class="modal-content">
            <span class="close" id="closeModalBtn">&times;</span>
            <h2>Detail Postingan</h2>
            <p>Kode Pendonor: <span id="kodePendonor"></span></p>
            <p>Nama: <span id="nama"></span></p>
            <p>Tanggal Posting: <span id="tanggalPosting"></span></p>
            <p>Konten:</p>
            <img id="gambar" src="" alt="Gambar">
            <p id="teks"></p>
        </div>
    </div>

    <script src="script.js"></script>
</body>
</html>

CSS (style.css):

css
/* Style modal container */
.modal {
    display: none;
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background-color: rgba(0, 0, 0, 0.7);
}

/* Style modal content */
.modal-content {
    background-color: #fff;
    margin: 10% auto;
    padding: 20px;
    border: 1px solid #888;
    width: 70%;
    max-width: 600px;
    position: relative;
}

/* Style close button */
.close {
    position: absolute;
    top: 0;
    right: 0;
    padding: 10px;
    cursor: pointer;
}

/* Style modal content elements */
h2 {
    margin-top: 0;
}

img {
    max-width: 100%;
}

/* Hide scrollbar for modal */
body {
    overflow: hidden;
}

JavaScript (script.js):

javascript
// Function to open modal
function openModal() {
    var modal = document.getElementById("infopostingan");
    modal.style.display = "block";
}

// Function to close modal
function closeModal() {
    var modal = document.getElementById("infopostingan");
    modal.style.display = "none";
    document.body.style.overflow = "auto";
}

// Assign click event handlers to open and close modal buttons
document.getElementById("openModalBtn").addEventListener("click", openModal);
document.getElementById("closeModalBtn").addEventListener("click", closeModal);

// Function to set modal content based on status details
function setModalContent(status) {
    document.getElementById("kodePendonor").textContent = status.kode_pendonor;
    document.getElementById("nama").textContent = status.nama;
    document.getElementById("tanggalPosting").textContent = status.tanggal_posting;
    document.getElementById("gambar").src = status.gambar;
    document.getElementById("teks").textContent = status.teks;
}

// Example status data
var statusDetail = {
    kode_pendonor: "12345",
    nama: "Nama Pendonor",
    tanggal_posting: "27 Oktober 2023",
    gambar: "gambar.jpg",
    teks: "Ini adalah teks status."
};

// Set modal content when the page loads
setModalContent(statusDetail);

Jika URL gambar berada dalam database dan Anda ingin menampilkannya dalam elemen HTML, Anda dapat mengganti atribut src dengan URL yang disimpan dalam database. Anda perlu mengambil URL gambar dari kolom yang sesuai dalam basis data dan menggantinya dalam tag <img>.

Misalnya, jika URL gambar disimpan dalam kolom url_gambar dalam database dan Anda ingin menampilkannya dalam HTML, Anda dapat melakukannya seperti ini:

html
<td>
    <img src="{{ $row->url_gambar }}" alt="" style="width:100px; height:100px;">
</td>

Dengan melakukan ini, gambar akan ditampilkan berdasarkan URL yang disimpan dalam kolom url_gambar pada setiap baris data dalam database Anda. Pastikan data URL gambar dalam basis data benar dan valid.

Unmatched ')' maksudnya ?
1 / 1

Kesalahan "Unmatched '('" muncul ketika ada tanda kurung (biasanya kurung tutup )) yang tidak memiliki pasangan atau tidak diakhiri dengan benar dalam kode Anda. Ini adalah kesalahan sintaksis yang dapat terjadi dalam bahasa pemrograman, markup HTML, atau bahasa lainnya.

Dalam konteks kode yang Anda berikan, "Unmatched '('" mungkin mengacu pada kurung buka ( yang terdapat dalam atribut src tag <img>. Ini seharusnya tidak menyebabkan kesalahan jika terdapat pasangan kurung tutupnya dengan benar.

Namun, jika Anda mendapatkan pesan kesalahan "Unmatched '('" dengan kode tertentu, mungkin ada kesalahan lain dalam kode yang memicu pesan kesalahan ini. Pastikan atribut src tag <img> diisi dengan benar dan tidak ada kesalahan sintaksis di sekitarnya. Jika Anda dapat memberikan contoh kode yang lebih lengkap, saya dapat membantu Anda lebih lanjut untuk mengidentifikasi masalahnya.

<input class="kolom form-control" type="text" placeholder="" readonly> memasukkan gambar ke placeholder apakah bisa ?
1 / 1

Anda tidak bisa langsung memasukkan gambar ke dalam atribut placeholder dari elemen input <input type="text">. Atribut placeholder digunakan untuk menampilkan teks panduan yang akan hilang saat pengguna mulai memasukkan teks ke dalam input.

Jika Anda ingin menampilkan gambar di sebelah atau di atas input, Anda dapat menggunakan elemen <img> atau elemen lain untuk menampilkan gambar tersebut. Sebagai contoh:

html
<img src="URL_GAMBAR" alt="Gambar" style="width: 100px; height: 100px;">
<input class="kolom form-control" type="text" placeholder="Teks Panduan" readonly>

Dalam contoh di atas, gambar akan ditampilkan di atas input dan input akan memiliki teks panduan di dalam atribut placeholder. Anda hanya perlu mengganti "URL_GAMBAR" dengan URL gambar yang ingin Anda tampilkan.

Jika Anda ingin membuat gambar dalam modal mengisi lebar modal tetapi tidak terlalu tinggi, Anda dapat mengatur lebar gambar agar sesuai dengan lebar modal dan membatasi tinggi gambar dengan CSS. Anda juga dapat mengatur max-height agar gambar tidak melebihi tinggi tertentu. Berikut perubahan yang perlu Anda lakukan pada kode Anda:

  1. Atur lebar gambar agar sesuai dengan lebar modal (misalnya, 100% dari lebar modal).
  2. Atur max-height agar gambar tidak melebihi tinggi tertentu.

Contoh HTML dan CSS yang diubah:

html
<div class="modal fade" id="infopostingan{{ $row->id }}" tabindex="-1" role="dialog" aria-labelledby="exampleModalLabel" aria-hidden="true">
    <div class="modal-dialog modal-lg" role="document">
        <div class="modal-content">
            <!-- ... (bagian lainnya) ... -->
            <div class="modal-body">
                <!-- ... (bagian lainnya) ... -->
                <div class="form-group" style="text-align: center;">
                    <img src="{{ $row->gambar }}" alt="" style="width:100%; max-height: 400px;"> <!-- Sesuaikan max-height dengan tinggi yang Anda inginkan -->
                </div>
                <!-- ... (bagian lainnya) ... -->
            </div>
        </div>
    </div>
</div>

Dalam contoh di atas, gambar akan mengisi seluruh lebar modal, dan tinggi gambar akan dibatasi maksimum hingga 400 piksel. Anda dapat mengganti nilai max-height dengan nilai yang sesuai dengan keinginan Anda. Dengan cara ini, gambar akan terlihat baik pada modal yang lebih lebar tetapi tidak terlalu tinggi.
